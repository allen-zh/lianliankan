[快来体验吧](http://snowandmumu.github.io/lianliankan/dest)

## 如何构建代码

签出项目：

```bash
git clone https://github.com/snowandmumu/lianliankan.git
```

安装依赖

```bash
npm install
```

安装gulp

```bash
npm -g install gulp
```

编译运行

```bash
gulp serve
```
